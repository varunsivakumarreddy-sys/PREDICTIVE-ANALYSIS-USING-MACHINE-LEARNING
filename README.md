## 📌 Overview

This project focuses on predicting house prices using Machine Learning techniques. The dataset contains various features such as location, size, number of bedrooms, and other property details.

The goal is to build a regression model that can accurately estimate house prices based on input features.

---

## 🎯 Objective

To develop a machine learning model that predicts house prices using regression algorithms and evaluates its performance.

---

## 🧩 Key Areas Covered

• Data preprocessing and cleaning
• Handling missing values
• Feature selection
• Model training using regression
• Model evaluation and performance metrics
